# Desafio: Criando o Crachá do Personagem com toString

## Cenário

Vamos criar o "crachá" do seu personagem.

## Sua Missão

1. Vá na classe Personagem
2. Implemente o método `toString`
3. Faça ele retornar uma frase bonita, por exemplo: "Nome: [nome] | Força: [forca]"
4. No Main, apague os prints antigos e coloque apenas:
   - `System.out.println(heroi);`
   - `System.out.println(oponente);`

Veja a mágica acontecer no console! ✨

## Dica

O método toString deve retornar (return) uma String formatada com os dados do objeto. O Java chamará esse método automaticamente quando você usar o objeto dentro do println.
