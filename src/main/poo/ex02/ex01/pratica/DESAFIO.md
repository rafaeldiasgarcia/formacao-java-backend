# Desafio: Criando sua Primeira Classe e Objeto

## Cenário

Você precisa criar um sistema básico de personagens para um jogo de RPG.

## Sua Missão

1. Crie uma classe chamada `Personagem`
2. Ela deve ter dois atributos:
   - `nome` (texto)
   - `forca` (número inteiro)
3. Crie um Construtor para preencher esses dois dados
4. No método `main`, crie um objeto chamado `heroi` com nome "Aragorn" e força 100
5. Imprima na tela: "O heroi Aragorn tem força 100"

## Dica

Use `System.out.println` concatenando com `+`.
