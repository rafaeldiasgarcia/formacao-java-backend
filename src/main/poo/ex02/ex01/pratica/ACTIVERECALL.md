# Active Recall - Classes e Objetos

## Perguntas e Respostas

### Questão 1: Classe vs Objeto

**Pergunta:** (Verdadeiro ou Falso) A Classe é o objeto real que ocupa memória e roda no programa.

**Resposta:** Falso

---

### Questão 2: Palavra-chave para Criar Objetos

**Pergunta:** (Múltipla Escolha) Para criar um novo objeto a partir de uma classe, qual palavra-chave (keyword) obrigatória devemos usar?

a) create

b) make

c) new

d) class

**Resposta:** c) new

---

### Questão 3: Nome do Construtor

**Pergunta:** (Verdadeiro ou Falso) O nome do método Construtor deve ser sempre idêntico ao nome da Classe.

**Resposta:** Verdadeiro
