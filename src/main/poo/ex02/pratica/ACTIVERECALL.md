# Active Recall - Getters e Setters

## Perguntas e Respostas

### Questão 1: Função do Setter

**Pergunta:** (Verdadeiro ou Falso) O método Setter serve para ler/pegar o valor de um atributo.

**Resposta:** Falso

---

### Questão 2: Validação no Setter

**Pergunta:** (Múltipla Escolha) Por que colocamos um if dentro do Setter?

a) Para deixar o código mais bonito.

b) Para validar o dado antes de alterar o atributo, impedindo valores inválidos.

c) Para criar um novo objeto.

**Resposta:** b) Para validar o dado antes de alterar o atributo, impedindo valores inválidos.
