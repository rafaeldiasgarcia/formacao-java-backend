# Active Recall - Atributos e Métodos Estáticos (static)

## Perguntas e Respostas

### Questão 1: Compartilhamento de Variáveis Static

**Pergunta:** (Verdadeiro ou Falso) Se eu alterar o valor de uma variável static em um objeto, o valor muda para todos os outros objetos dessa classe.

**Resposta:** Verdadeiro

---

### Questão 2: Acesso a Variáveis Estáticas

**Pergunta:** (Múltipla Escolha) Qual a maneira recomendada de acessar uma variável estática chamada contador da classe Jogo?

a) meuJogo.contador

b) Jogo.contador

c) this.contador

**Resposta:** b) Jogo.contador

---

### Questão 3: Uso de this em Métodos Static

**Pergunta:** (Verdadeiro ou Falso) Posso usar this.nome dentro de um método public static void....

**Resposta:** Falso
