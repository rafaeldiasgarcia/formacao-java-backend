### Questão 1: Identificação Automática

**Pergunta:** (Verdadeiro ou Falso) O Java escolhe qual método usar automaticamente, analisando os tipos de dados que eu passo nos parênteses.

**Resposta:** Verdadeiro

---

### Questão 2: Criando uma Sobrecarga Válida

**Pergunta:** (Múltipla Escolha) Se eu tiver o método `correr(int velocidade)` e quiser criar uma sobrecarga, qual das opções abaixo é válida?

a) `void correr(int velocidade)`

b) `void correr(String terreno)`

c) `int correr(int velocidade)`

**Resposta:** b) `void correr(String terreno)`
