# Desafio: O Estoque da Loja

## Cenário

Você precisa criar um pequeno sistema para consultar preços de produtos usando Map (HashMap), pois queremos buscar pelo nome do produto.

## Sua Missão

1. Crie um `Map<String, Double>` chamado `estoque`
2. Adicione 3 produtos:
   - "Celular" custando 1500.00
   - "Mouse" custando 50.00
   - "Teclado" custando 120.00
3. Imprima o preço do "Mouse" usando o comando correto de busca

## Dica

Lembre-se que para inserir usamos `.put(chave, valor)` e para buscar usamos `.get(chave)`.
