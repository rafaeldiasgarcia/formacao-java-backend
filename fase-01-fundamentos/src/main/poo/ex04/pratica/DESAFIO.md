# Desafio: Sobrecarga de Métodos

## Cenário

Vamos dar mais opções de combate ao seu herói.

## Sua Missão

1. Mantenha o seu método `atacar(Personagem oponente)` como está
2. Crie um novo método `atacar` sem parâmetros
3. O que ele faz: Imprime "O [nome] está esperando... mas não atacou ninguém."
4. No Main, chame `heroi.atacar();` (sem nada dentro) antes de chamar o ataque real contra o Orc

## Dica

Você terá dois métodos com o mesmo nome, mas com parâmetros diferentes. O Java escolherá automaticamente qual executar baseado no que você passar nos parênteses.
