## Questão 1: Do-While
**Pergunta:** Sobre o laço do-while, é correto afirmar que:

a) Ele só executa se a condição for verdadeira logo de cara.  
b) Ele repete o código infinitamente sem verificar condições.  
c) Ele é usado exclusivamente para arrays.  
d) Ele executa o bloco de código pelo menos uma vez, mesmo que a condição seja falsa no início.

**Resposta:** d) Ele executa o bloco de código pelo menos uma vez, mesmo que a condição seja falsa no início.

## Questão 2: Operador Lógico &&
**Pergunta:** No nosso código do banco, alteramos a condição de saque para `if (saldo >= saque && saque > 0)`. Qual o objetivo do operador `&&` aqui?

a) Garantir que O SALDO seja suficiente E que o valor do saque seja positivo ao mesmo tempo.  
b) Criar um loop infinito se o saldo for insuficiente.  
c) Somar o valor do saldo com o valor do saque.  
d) Permitir que o saque ocorra se APENAS uma das condições for verdadeira.

**Resposta:** a) Garantir que O SALDO seja suficiente E que o valor do saque seja positivo ao mesmo tempo.

## Questão 3: Índices de Arrays
**Pergunta:** Se você cria um array com `new String[100]`, qual é o índice da ÚLTIMA posição válida onde você pode guardar algo?

a) 1  
b) 99  
c) 100  
d) 0

**Resposta:** b) 99
