# Conceito: Do-While (Fazer... Enquanto)

## Analogia

Imagine que você vai comer um bombom de uma caixa sortida.

- **While:** Você olha o bombom antes. Se parecer ruim, nem come. (Verifica → Faz).
- **Do-While:** Você come o bombom primeiro. Depois decide se quer comer outro ou parar. (Faz → Verifica).

## Regra de Ouro

O do-while garante que o código rode **pelo menos uma vez**, não importa a condição.
