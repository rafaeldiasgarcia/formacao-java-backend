# Desafio: Sistema de Decolagem de Foguete

## Cenário

Vamos criar um "Sistema de Decolagem de Foguete".

## Sua Missão

Crie um código que use um loop para fazer uma contagem regressiva de 10 até 1. Quando o loop acabar, imprima "Decolar!".

## Input (Entrada)

Nenhuma (o código já começa com o valor 10).

## Output (Saída esperada)

```
10
9
... (números intermediários)
1
Decolar!
```

## Dica

No loop for, em vez de aumentar (i++), você pode diminuir (i--).
