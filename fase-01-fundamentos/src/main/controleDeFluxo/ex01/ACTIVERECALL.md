# Active Recall - Controle de Fluxo

## Questão 1: Loop While
**Pergunta:** Verdadeiro ou Falso: O loop while executa o código pelo menos uma vez, mesmo que a condição seja falsa logo de cara.

**Resposta:** FALSO

---

## Questão 2: Switch Statement
**Pergunta:** Qual comando usamos quando queremos verificar várias opções simples de um único valor (ex: opções de um menu 1, 2 ou 3)?

a) if / else  
b) switch  
c) for

**Resposta:** b) switch

---

## Questão 3: Loop Infinito
**Pergunta:** O que acontece se você esquecer de colocar a linha que muda a variável (ex: i++ ou bateria - 5) dentro de um loop while?

a) O código dá erro e não roda.  
b) O loop roda apenas uma vez.  
c) Cria-se um "Loop Infinito" (o programa trava rodando para sempre).

**Resposta:** c) Cria-se um "Loop Infinito"
